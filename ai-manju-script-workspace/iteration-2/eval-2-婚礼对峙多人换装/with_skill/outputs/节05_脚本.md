# 镜头 01

## 时长
5 秒

## 画面描述
【大远景，平视；镜头不动】无人物，锦云会酒店大门外台阶与门廊全貌，台阶两侧红毯，远处一列白色奔驰依次驶入，雨后湿亮街面。

## 对白
无

## 画风
韩漫风 + 都市虐心半写实；线条克制，五官接近写实但保留漫画化光泽，东亚面孔，皮肤质感细腻有反光；色彩低饱和电影感，常用青橙对比与暖灰统调，柔焦+暖色光源；构图大量留白，常见斜下俯视与背影；整体接近电影海报质感，2D 平面感为主关键道具有体积感。

## 场景
锦云会酒店大门外台阶，午后阴天约 13:30，半室外：宽石阶共九级 + 两侧红毯 + 门廊白光灯（暖白）+ 黑色铸铁门把 + 远处接待奔驰车队 + 雨后湿亮街面 + 大门玻璃旋转门。

## 光影氛围
阴天散光 + 门廊暖白边缘光；色调暖灰湿亮，氛围克制肃穆、压着大事将至。
