# 节 08：摔门

苏念踹开顾衡办公室的门。

整间四十平的玻璃景观办公室在一瞬间凉了下来——下午两点的阳光被她带进来又被她甩出去。落地窗外的 CBD 看上去像一张静止的图。

顾衡正坐在办公桌后审一份合同，闻声抬眼。他没说话，只是把手里的派克笔轻轻搁回桌面，金边眼镜后的眼神冷下来一分。

苏念三步走到办公桌前，把抓在手里的那一沓离婚协议——左上角已经被她攥得皱了——啪地拍在桌面上。一张纸滑出来，飘到顾衡面前。

"签。"她说。

顾衡盯着那沓纸，沉默了三秒，才把目光抬起来落在她脸上。
"苏念，"他开口，"再聊聊。"

苏念已经转过身，朝门口走。她的高跟靴踩在抛光大理石上，一步比一步重。

走到门口，她回头扔了一句：
"我不会再回来了。"

她甩门出去。
办公室的玻璃门被震得颤了一下，又缓缓合上。
桌上那张飘出来的纸停在了顾衡的咖啡杯旁边——是协议的最后一页，落款栏空着。
